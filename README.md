# Comparison In-Memory Caching

## Requirements

- Redis
- [Redis client for Go](https://pkg.go.dev/github.com/go-redis/redis/v9)

## Installation

```shell
go install github.com/jujube5/testmemorymain
```

```shell
go get github.com/jujube5/testmemory
```

## Quickstart

```shell
cd testmemorymain
```
```shell
go run main.go
```
